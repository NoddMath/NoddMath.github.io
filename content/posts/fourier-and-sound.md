一段声音看起来可以极其复杂，但傅里叶的想法是：复杂并不一定需要复杂的基本元素。足够多的简单振动叠加起来，就能重新构造几乎任何波形。

## 把波拆开

对周期为 $2\pi$ 的函数，傅里叶级数写成

$$
f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty}\left(a_n\cos nx + b_n\sin nx\right).
$$

每一项都是一种纯粹频率。系数 $a_n$ 与 $b_n$ 告诉我们，这个频率在原始信号中占了多少。

如果把声音比作一种颜色，那么基频像主色，高次谐波则决定了它是小提琴、长笛，还是人的嗓音。

## 方波的意外

方波有尖锐的转折，而正弦波处处光滑。它们似乎不该有亲缘关系，但加入越来越多的奇次谐波后，光滑的正弦波会逐渐逼近方波：

$$
\operatorname{square}(x) \approx \frac{4}{\pi}\left(\sin x + \frac{\sin 3x}{3} + \frac{\sin 5x}{5} + \cdots\right).
$$

转折附近会出现不会完全消失的过冲，这叫作 Gibbs 现象。它提醒我们：收敛不仅关乎“是否接近”，也关乎“以什么方式接近”。

```js
function squareWave(x, terms = 12) {
  let y = 0;
  for (let k = 0; k < terms; k++) {
    const n = 2 * k + 1;
    y += Math.sin(n * x) / n;
  }
  return (4 / Math.PI) * y;
}
```

## 从分析到聆听

频谱图把时间中的声音翻译成频率中的结构。翻译之后，原本藏在波形里的和声、噪声和节奏变得清晰。傅里叶变换并没有创造新信息，它只是选择了一组更适合回答某些问题的坐标。

数学最动人的时刻之一，就是一个抽象的基底展开，最后真的改变了我们听见声音的方式。
